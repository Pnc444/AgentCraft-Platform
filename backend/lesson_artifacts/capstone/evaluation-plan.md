# Capstone Evaluation Plan

## Quiz layer
- Check terminology and conceptual understanding.

## Verifier layer
- Confirm artifact existence, config shape, required fields, and audit outcomes.

## Human review layer
- Compare release candidates for clarity, usefulness, and rollout judgment.

## LLM judge layer
- Score free-form deliverables against the helpful, truthful, harmless rubric.

## Release rule
- No single layer is enough by itself. Use the method that matches the evidence you need.
