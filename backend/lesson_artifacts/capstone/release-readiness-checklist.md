# Capstone Release Readiness Checklist

- The automation brief names a trigger, tools, review boundary, and receipt path.
- Guardrail claims map to runtime controls or review steps.
- The permission matrix classifies runtime and control-plane actions.
- Evaluation cases cover positive, negative, adversarial, and regression paths.
- The evaluation plan combines quizzes, verifiers, human review, and LLM judging.
- Validation passes before publish.
