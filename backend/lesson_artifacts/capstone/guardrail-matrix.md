# Guardrail Matrix

| Goal | Concrete control | Evidence |
| --- | --- | --- |
| Helpful | Review boundary keeps the workflow moving instead of dead-ending | Approval step is documented |
| Truthful | Unsupported claims are labeled unknown | Rubric or verifier checks citation/evidence quality |
| Harmless | Untrusted channels stay on pairing; risky tools are denied or sandboxed | Config + audit output |
| Operational | Decisions produce receipts | Log or review note exists |
