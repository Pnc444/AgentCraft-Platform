# One Automation, Drawn as a Sequence

The Monday audit, numbered. Notice where the human sits (step 4) and where the record gets written (step 5).

1. **Trigger** — Monday, 9:00. The schedule fires; nobody presses anything.
2. **Work** — the assistant runs the security audit and gathers findings.
3. **Shape** — findings become a short, ordered checklist (worst first).
4. **Review** — the checklist waits for a person. Changes only happen after a human yes.
5. **Receipt** — the report and the decision are saved, dated.
6. **Stop** — the run ends. The next one is next Monday's business, not tonight's.

Every trustworthy automation is this same sequence with different nouns: something starts it, bounded work happens, a person holds the irreversible step, paper gets left behind, and it ends.
