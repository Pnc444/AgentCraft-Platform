# First Rollout — the whole thing fits on an index card

1. Connect **one** channel — the app you actually use most: ______________
2. Leave the DM policy on `pairing`. (It already is. Leaving it alone is the correct move.)
3. Approve exactly one person — you: `openclaw pairing approve <channel> <code>`
4. Send yourself a message. Watch it answer.

**Done.** That is a complete, correct first rollout — not a starter version of one.

Widening later (a second person, a group room) is just step 3 again with a new code, whenever you choose. There is no pressure to widen anything today, or ever. If a group room ever joins: turn on `requireMention` so the assistant only speaks when spoken to.
