# The Agent Map — six questions, one helper

Any agent can be described by six plain questions. Juno's answers sit beside each one.

| Question | What it asks | Juno's answer |
| --- | --- | --- |
| Goal: | What does the human actually want? | Three good articles about sleep and memory, summarized |
| Plan: | What should it try next? | Start with a search; adjust if results are junk |
| Tools: | What is it allowed to do? | Search and read — nothing else |
| Memory: | What should it keep track of? | The articles found and kept so far |
| Check: | How does it know the work is good? | "Is this article actually about sleep and memory?" |
| Stop: | How does it know it is finished? | Three good summaries delivered → stop |

---

You never need to memorize this map. Every lesson in Module 4 reuses it, so it settles in from use. If you ever want it again, it lives right here.
